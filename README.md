# Sistema de Votación

Proyecto web desarrollado con .NET y Angular que permite
registro de usuarios, autenticación, votación, evita voto duplicado, crear candidato, visualizar reportes, elimminar candidatos sin votos, ver auditoria de votatantes, portal de admin y de user.

## Tecnologías
- .NET 8
- Angular
- Firebase Firestore
- JWT

## Configuración
Para ejecutar el proyecto se debe configurar Firebase
mediante una cuenta de servicio local (no incluida en el repositorio).
